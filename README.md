# Help-and-Support


Last updated: 02 January 2026

Welcome to the Help & Support page for PHOC. This section helps you understand how the app works and assists you in resolving common issues.

# About PHOCccccc

PHOC is a color transfer application that allows you to apply cinematic color styles between images and videos using server-side processing.

PHOC supports the following features:
• Image to Image color transfer
• Video to Video color transfer
• Image to Video color transfer
• Video to Image color transfer

# How to Use PHOC

1. Select an image or video from your device.
2. Select the target image or video for color transfer.
3. Tap on the process button.
4. Wait while the media is processed on the server.
5. Save the final result to your device.

# Internet Requirement

PHOC uses server-side processing to deliver high-quality results. An active internet connection is required to upload media files to the server and receive the processed output. The app cannot be used offline.

# Permissions

PHOC requires access to your device’s media storage to allow you to select images and videos.
Internet permission is required to upload media files for server-side processing.

# Saving Results

Currently, PHOC allows users to save the processed images and videos directly to their device. Sharing options may be added in future updates.

# Common Issues & Troubleshooting

• Large images may take 1–2 seconds to load on the second screen. This is expected behavior and will be optimized in future updates.
• Processing time depends on file size, video length, and internet speed.
• If the app appears slow or unresponsive, please ensure a stable internet connection and sufficient device storage.

# Frequently Asked Questions (FAQs)

Q: Why does video processing take more time?
A: Video files are larger and require more server resources to process.

Q: Where is my processed image or video saved?
A: The final output is saved directly to your device storage.

Q: Can I use PHOC without an internet connection?
A: No, an internet connection is required for server-side processing.
